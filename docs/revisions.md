## Revision 1.0
First version and POC for many details.

![Revision 1.0 Board](images/revison_1_board.png)

### **Assembly Guide**: [Revision 1.0 Assembly Instructions](#assembly-rev-10)

### **Details**:
- CH32V003 debug pins
- Relay debug pins
- CH32V003 Input Pins
- Protoboard for debugging
- Relay sound POC
- Buzzer sound POC
- I2C interface POC
- Button POC
- 3D Printed CR2032 battery holder POC

### **Known Issues**:
- Silkscreen issues / inconsistencies
- Relay footprint incorrect for intended relay (replacements avalible)
- Buzzer footprint incorrect
- Diode footprint incorrect
- Can not be programmed with SAO Header
